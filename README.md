# Space Mosquitoes

Canvas experiments with vanilla js and HTML

## Run locally

```
git clone git@github.com:tadast/space_mosquitos.git
cd space_mosquitos
ruby -rwebrick -e'WEBrick::HTTPServer.new(:Port => 5555, :DocumentRoot => Dir.pwd).start'
```

visit http://localhost:5555
