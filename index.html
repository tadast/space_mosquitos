Micro cosmos

Every time you interact with the screen inside of an experiment new particles are generated. Each particle is coded individually to follow certain rules to move around their 2D world.

Each experiment has random variables that influence either distribution or movement of each particle. This means every time you run an experiment it generates a unique, never before seen 2D world!

Choose a micro universe:

  • Galaxy
  • Chaos
  • Warp speed
  • Colour fill
